# Praktika proovitöö kokkuvõte

Proovitöö tegemiseks kulus ~15-20 tundi. Suuremaid probleeme oli ainult 2:
1. jäin kinni kuidas laudadele toole juurde lisada, mille ma lihtsalt lõpuks tegin taustaks koos staatiliste laudadega (tahtsin teha ümberliigutatavad lauad, aga aeg sai otsa kuna alustasin tegelemisega liiga hilja). 
2. Ajavööndite tõttu tekkinud erinevused, mille parandasin kindla offsetiga.

Projektis on kommenteeritud ära kus on kasutatud suurel määral AI abi või internetist võetud koodi.
